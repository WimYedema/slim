# Security Policy

## Scope

Slim is a client-side single-page application. All data is stored in the user's browser (localStorage). The only network communication is optional P2P board sharing via public Nostr relays, encrypted with AES-256-GCM.

## Reporting a Vulnerability

If you discover a security issue, please report it privately:

- **Email**: wim.yedema@gmail.com
- **Subject**: `[SECURITY] Slim -- <brief description>`

Please do not open a public issue for security vulnerabilities.

## Response

I will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.
